## 2024-05-24 - FlowEditor Icon Buttons Missing ARIA Labels\n**Learning:** When adding multiple action buttons (like delete, duplicate, play) represented by icons in a form or node editor without visible text, they often lack ARIA labels, rendering them inaccessible to screen readers. This pattern is easily missed during initial development but significantly affects usability for users relying on assistive technologies.\n**Action:** Always include an `aria-label` attribute on buttons that contain only icons. Validate accessibility by ensuring every interactive element has an accessible name, especially in complex editors with many inline actions.
